# SH Solar Energy Website

## קבצים
- `index.html` — האתר הראשי
- `style.css` — כל העיצוב
- `script.js` — אנימציות ותפריט
- `images/` — תיקייה להוספת תמונות (תיצור ידנית)

## הוספת תמונות פרויקטים

1. צור תיקייה `images/` באותו מיקום כמו index.html
2. העתק תמונות לתיקייה (jpg, webp מומלץ)
3. ב-index.html, חפש את ה-comments שכתוב בהם:
   `<!-- REPLACE WITH: <img src="images/project1.jpg" ... /> -->`
4. החלף את בלוק ה-placeholder-img בשורת ה-img המתאימה

## Deploy ל-Vercel (חינם)

1. צור חשבון ב-vercel.com
2. גרור את תיקיית הפרויקט לממשק Vercel
3. לחץ Deploy
4. האתר באוויר תוך דקה!

## חיבור דומיין מ-GoDaddy ל-Vercel

1. ב-Vercel: Settings → Domains → Add Domain
2. הכנס: shsolarenergy.com
3. Vercel יתן לך 2 רשומות DNS להוסיף
4. ב-GoDaddy: Manage DNS → הוסף את הרשומות
5. תוך 10-30 דקות האתר עולה על הדומיין שלך
